# SingBoxGUI

## English | 🇬🇧

SingBoxGUI is a graphical user interface for [Sing-box](https://sing-box.sagernet.org/), designed to simplify its configuration and management.

### Features:
- Start and stop Sing-box with one click.
- Manage subscription URLs and update configurations from the internet.
- Enable or disable system proxy settings.
- View real-time logs in an embedded WebView.
- Minimize to system tray with quick access controls.
- Automatically clear proxy settings upon exit.

### Installation:
1. Download the latest release from the [GitHub Releases](https://github.com/BrontoDIY/SingBoxGUI/releases) page.
2. Extract the contents of the zip file.
3. Run `SingBoxGUI.exe`.

### Requirements:
- Windows 10/11 (64-bit)
- .NET Framework 4.8.1 or higher
- WebView2 Runtime installed

## فارسی | 🇮🇷

SingBoxGUI یک رابط گرافیکی برای [Sing-box](https://sing-box.sagernet.org/) است که برای ساده‌سازی مدیریت و پیکربندی آن طراحی شده است.

### ویژگی‌ها:
- راه‌اندازی و توقف Sing-box تنها با یک کلیک.
- مدیریت لینک‌های اشتراک و به‌روزرسانی تنظیمات از اینترنت.
- فعال یا غیرفعال کردن تنظیمات پروکسی سیستم.
- مشاهده لاگ‌ها به‌صورت زنده در WebView داخلی.

- پاکسازی خودکار تنظیمات پروکسی هنگام خروج.
